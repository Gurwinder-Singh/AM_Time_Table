# AM_Time_Table
 Andromeda Time Table Manager

## Prerequisites
For Run this project You need:-
```
Java 8
NetBeans IDE 8.0
```

## Build
```
Write Click on Project in NetBeans -> Package As ->Installers or Zip Distribution
```
